# NotesApp  

A secure full-stack notemaking application built with PERN stack (PostgreSQL, Express, React, Node).  

## Features  
- Secure note management  
- Full-stack PERN architecture  

## Tech Stack  
- **Frontend**: React  
- **Backend**: Express, Node  
- **Database**: PostgreSQL  

## Installation  
1. Clone the repository  
2. Install dependencies for both client and server:  
   ```  
   cd client && npm install  
   cd ../server && npm install  
   ```  

## Usage  
1. Start the client:  
   ```  
   cd client && npm run dev  
   ```  
2. Start the server:  
   ```  
   cd server && nodemon run dev  
   ```
